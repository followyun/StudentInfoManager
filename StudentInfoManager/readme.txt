该项目为慕课网学习项目学生信息管理系统
总体框架为hibernate+struts2+mysql
运行平台：win10
运行环境：apache Tomcat7.0
开发工具：Eclipse Neon 4.6.2, Navicat
实现功能为：
1.管理员登陆
2.登陆成功可以查看所有学生信息
3.能够增加学生信息，修改学生信息，删除学生信息。

使用方法：
1.在Eclipse中导入该工程，
2.在mysql中创建studentinfomanager数据库，并执行WebRoot/sql/test.sql文件中的sql语句。
2.Run As->Run on Server
3.运行成功后，在浏览器地址栏输入:localhost:8080/StudentInfoManager即可进入登陆页面
